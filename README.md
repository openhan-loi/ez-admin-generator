# EzAdmin Upload Generator - 재고 DB 동기화

패킹리스트 파일을 분석하여 제품명, 색상, 사이즈, 수량을 추출하고 이지어드민 업로드 파일을 생성하는 웹 기반 시스템입니다.

## 📋 현재 구현 상태 (1단계)

### ✅ 완료된 기능

1. **모던한 웹 UI**
   - 프리미엄 디자인 시스템 (그라데이션, 애니메이션)
   - 4단계 진행 표시기
   - 반응형 레이아웃
   - 토스트 알림 시스템

2. **도매인 관리**
   - 도매인 등록 및 삭제
   - LocalStorage를 통한 영구 저장
   - 드롭다운에서 선택 가능
   - 태그 형태로 시각화

3. **파일 업로드**
   - 드래그 앤 드롭 지원
   - 파일 선택 버튼
   - 파일 형식 검증 (.xlsx, .xls, .csv)
   - 파일 크기 제한 (10MB)

4. **패킹리스트 분석**
   - 자동 헤더 감지
   - 컬럼 자동 매칭 (제품명, 색상, 사이즈, 수량)
   - 진행률 표시
   - 분석 결과 테이블 표시

### 📌 다음 단계 (2-4단계)

2. **이지어드민 데이터 매핑**
   - 전체 제품 리스트 업로드
   - 자동 매칭 알고리즘

3. **수동 매핑 UI**
   - 매칭 안 된 제품 표시
   - 수동 선택 인터페이스

4. **업로드 파일 생성**
   - 이지어드민 형식으로 변환
   - 다운로드 기능

## 🚀 실행 방법

### 로컬에서 실행 (현재)

1. 터미널에서 프로젝트 폴더로 이동:
```bash
cd "C:\Users\day\Documents\n8n\Upload Generator"
```

2. Python HTTP 서버 실행:
```bash
python -m http.server 8080
```

3. 브라우저에서 접속:
```
http://localhost:8080
```

### 테스트 방법

1. **도매인 등록**
   - "새 도매인 등록" 버튼 클릭
   - 도매인 이름 입력 (예: "도매1", "ABC도매")
   - "저장" 버튼 클릭

2. **패킹리스트 분석**
   - 도매인 선택
   - 패킹리스트 파일 업로드 (샘플: `sample_packing_list.xlsx`)
   - "파일 분석하기" 버튼 클릭
   - 분석 결과 확인

## 🎨 특징

- **자동 컬럼 감지**: 엑셀 파일의 헤더를 자동으로 분석하여 제품명, 색상, 사이즈, 수량 컬럼을 찾습니다
- **유연한 형식**: 다양한 컬럼명을 지원합니다 (예: "제품명", "상품명", "품명" 모두 인식)
- **오프라인 저장**: 도매인 정보는 브라우저에 저장되어 다시 접속해도 유지됩니다
- **프리미엄 UI**: 모던한 디자인과 부드러운 애니메이션으로 사용자 경험 향상

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **라이브러리**:
  - XLSX.js (엑셀 파일 읽기)
  - Google Fonts (Inter 폰트)
- **저장소**: LocalStorage (브라우저)

## 📊 지원 파일 형식

- `.xlsx` (Excel 2007+)
- `.xls` (Excel 97-2003)
- `.csv` (쉼표로 구분된 값)

## 🐛 문제 해결

### 파일이 업로드되지 않을 때
- 파일 형식이 .xlsx, .xls, .csv인지 확인
- 파일 크기가 10MB 이하인지 확인

### 분석이 실패할 때
- 엑셀 파일의 첫 번째 행이 헤더인지 확인
- 필수 컬럼(제품명)이 포함되어 있는지 확인

### 도매인이 저장되지 않을 때
- 브라우저의 LocalStorage가 활성화되어 있는지 확인
- 시크릿 모드에서는 저장되지 않을 수 있음
